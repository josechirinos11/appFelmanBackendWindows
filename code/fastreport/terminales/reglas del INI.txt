CLAVE=VALOR
Clave es R + número de rectángulo + . + Id de perfil

En valor se guardan los siguientes datos separados por punto y coma
1-Código artículo… código sicar del artículo 
2-Tipo de artículo … perfil, accesorio, servicio, superficie, vidrio 1-2-3-4-5
3-Uso  … uso del perfil 10 marco, 20 hojas etc. si no tiene uso o no es perfil valor será 0
4-Situación … 1 superior, 2 derecha...etc. 0 no determinada o indicada
5-Descripción. Aquí tendremos la descripción técnica si la hay, sino la del artículo. 8
6-Acabado1 el acabado interior.
7-Acabado2 el acabado exterior si lo hay
8-LinkID si este artículo está vinculado a otro, aquí estará el id de perfil al que está vinculado.
9-Cantidad. Unidades por esta línea de descompuesto.
10-LON_A Largo para perfiles y gomas, ancho en caso de ser superficie
11-LON_B Alto para superficies. 
12-Número Clasificación del artículo (campo nuevo en articulos para su clasificación)
13-Texto Clasificación del artículo (campo nuevo en articulos para su clasificación)
14-IdFigura Id de la figura desde donde se generó el descompuesto.
